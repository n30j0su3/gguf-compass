¿Qué GPU tienes?

Selecciona tu tarjeta gráfica o la cantidad de VRAM disponible. Si no tienes GPU dedicada, selecciona "Sin GPU".

⚠️ Regla de Oro de Quantización
Q4_K_M es el mínimo para trabajo real. Por debajo de Q4, los modelos pierden coherencia, cometen errores factuales, y generan código con bugs sutiles. Solo usa Q3 o Q2 si:
  • Estás haciendo testing/experimentación (no producción)
  • Tu VRAM es ≤ 4GB y no tienes alternativa
  • El modelo es ≤ 3B parámetros (los pequeños toleran mejor quant bajo)
Para trabajo real (coding, agentes, producción): Q4_K_M o superior.
